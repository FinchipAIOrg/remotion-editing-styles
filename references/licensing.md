# Licensing and Attribution

This skill is an independently written workflow that is compatible with Remotion. It is not an official Remotion product.

## Remotion

The `remotion-dev/skills` repository currently points to the Remotion monorepo package and does not expose a separate repository license file. Remotion itself uses a custom license with eligibility rules for free use, a company-license requirement for some organizations, and restrictions on selling derivatives of Remotion code.

Before commercial redistribution or use:

1. Read the current Remotion license from the official repository.
2. Determine whether the user or organization qualifies for free use.
3. Obtain a company license when required.
4. Do not represent this skill or its starter as an official Remotion distribution.
5. Preserve third-party notices for packages and media.
